# IDR Landing Page Website

This project is a responsive and modern landing page developed for the Institute of Digital Risk (IDR). The website presents the organization’s vision, services, and contact section with a clean UI and smooth user experience.

## Features

- Responsive design for desktop, tablet, and mobile
- Modern navigation bar with logo
- Animated sections and smooth scrolling
- Service cards with hover effects
- Contact form UI
- Hero landing section with background image
- Mobile navigation menu

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Design
- CSS Animations
